Ejemplos de la propiedad Position

1. Position: Static (por defecto)

Static

Los elementos con position: static siguen el flujo normal del documento. Las propiedades top, right, bottom, left no tienen efecto.

2. Position: Relative

Static
Relative
Static

Position: relative mueve el elemento desde su posición original, pero conserva su espacio en el flujo del documento.

3. Position: Absolute

Contenedor con position: relative

Absolute

Position: absolute posiciona el elemento relativo al ancestro posicionado más cercano (no static). Si no encuentra ninguno, se posiciona relativo al body.
